JS Obfuscator Generator (Basic)

Bre, masukin kode JavaScript lo di bawah ini, nanti gue obfuscate biar agak puyeng bacanya!

Catatan Penting, Bro:

  • Ini cuma contoh dasar pake `javascript-obfuscator` versi CDN. Untuk proyek serius, disarankan instal via npm/yarn dan bundling.
  • Obfuscation bukan enkripsi. Kode yang diobfuscate masih bisa di-reverse engineer (walaupun lebih susah). Jangan pernah menyimpan informasi sangat sensitif (misalnya kunci API, password) langsung di JavaScript klien.
  • Beberapa fitur obfuscator bisa bikin kode lo lebih besar atau sedikit lebih lambat. Tes dulu ya!
